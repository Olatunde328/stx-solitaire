# ♠ STX Solitaire

STX Solitaire is a wallet-connected Klondike Solitaire game built on the Stacks blockchain.

## Live App

https://stx-solitaire.vercel.app

## Current Features

- Connect Stacks wallet
- Play Klondike Solitaire
- Track time, score, moves, and rewards
- Claim limited STX rewards on Stacks testnet
- Reward cooldown protection
- Player win tracking through Clarity contract
- Clean responsive UI

## Reward Model

The project is moving from unlimited rewards to a safer reward system:

- XP and wins for regular play
- Limited STX reward claims
- Daily cooldown protection
- Future leaderboard and tournament rewards

## Tech Stack

- Vite
- JavaScript
- Stacks Connect
- Clarity smart contract
- Vercel deployment
